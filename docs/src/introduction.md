# Introduction

This is the documentation for the `@freecodecamp/freecodecamp-os` package.

`freecodecamp-os` is a tool for creating and running interactive courses within Visual Studio Code.

The `freecodecamp-os` package is to be used in conjunction with the [freeCodeCamp - Courses](https://marketplace.visualstudio.com/items?itemName=freeCodeCamp.freecodecamp-courses) extension, for the optimal experience.
